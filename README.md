I created a flags database because I love flags. flagSorter.py creates a database of flags with 7 columns: id, colors, size, country, continent, symbols (e.g., weapons, people, animals, nature), and year adopted. Besides id and year, which have datatype INTEGER, the columns all have datatype TEXT. There is functionality for all 4 CRUD operations and a CLI to add new flags through 'ADD', view the flag database through 'VIEW', filter by flag attributes through 'FILTER', update flag columns through 'UPDATE', and delete flag rows through 'DELETE'. When adding and updating, there is further user input collected to identify every relevent flag attribute to work with. Only unique entries can be inserted, and there is error handling when invalid inputs are given to add, update, or delete. An interesting feature is that users can filter either for all colors in a flag, or just some/one (e.g., a filter for 'red, white' flags will return flags that have 'red, blue, white, green' as well as just 'red, white'). To run the app, simply run the program, and you will be prompted on how you'd like to interact with the database.
