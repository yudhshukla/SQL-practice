flagSorter.py creates a database of flags with 7 columns: id, colors, size, country, continent, symbols (e.g., weapons, people, animals, nature), and year adopted. There is functionality for all 4 CRUD operations and a CLI to add new flags, view the flag database, filter by flag columns, update flag columns, and delete flag rows. Only unique entries can be inserted, and there is error handling when invalid inputs are given to add, update, or delete. Additionally, users can filter either for all colors in a flag, or just some/one (e.g., a filter for 'red, white' flags will return flags that have 'red, blue, white, green' as well as just 'red, white').
